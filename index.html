Руководство по использованию zip2john

Инструкция по извлечению хешей из ZIP-архивов для John the Ripper

zip2john — утилита для извлечения криптографических хешей из ZIP-архивов, которые затем могут быть взломаны с помощью John the Ripper.

1. Основные команды

1.1 Базовое использование (один файл)

$ zip2john archive.zip > hash.txt

Эта команда извлечет хеш из archive.zip и сохранит его в файл hash.txt для последующего использования с John the Ripper.

1.2 Просмотр хеша в терминале

$ zip2john archive.zip

Хеш будет выведен прямо в терминал. Полезно для быстрой проверки содержимого.

1.3 Обработка нескольких ZIP-файлов

$ zip2john archive1.zip archive2.zip > hashes.txt

Или с использованием шаблонов:

$ zip2john *.zip > hashes.txt

2. Альтернативные вызовы

В некоторых дистрибутивах zip2john может быть скриптом или частью пакета John the Ripper.

2.1 Использование через Python-скрипт

$ python3 /usr/share/john/zip2john.py archive.zip > hash.txt

2.2 Прямой вызов из директории John

$ /usr/share/john/zip2john archive.zip > hash.txt

2.3 Прямой запуск John the Ripper на ZIP-архив

$ john --format=zip archive.zip
$ john --zip=archive.zip

3. Для RAR-архивов

$ rar2john archive.rar > hash.txt

4. Взлом хеша с помощью John the Ripper

4.1 Атака по словарю

$ john --wordlist=/usr/share/wordlists/rockyou.txt hash.txt

4.2 Атака по маске

$ john --mask='?l?l?l?l?l?d' hash.txt

4.3 Показать найденные пароли

$ john --show hash.txt

5. Полный рабочий пример

# Извлекаем хеш из защищенного архива
$ zip2john protected.zip > zip_hash.txt

# Запускаем атаку по словарю
$ john --wordlist=passwords.txt zip_hash.txt

# Показываем результаты
$ john --show zip_hash.txt

Важное предупреждение: Используйте эти инструменты только для:

  • Тестирования безопасности собственных архивов и систем
  • Восстановления забытых паролей к вашим собственным данным
  • Законных пентестов с явного разрешения владельца системы

Незаконное использование криптографических инструментов может повлечь юридическую ответственность.

6. Примечания и устранение проблем

6.1 Расположение утилит

zip2john обычно находится в одном из следующих каталогов:

  • /usr/sbin/
  • /usr/local/bin/
  • /usr/share/john/
  • /opt/john/run/

6.2 Проверка доступности

$ which zip2john
$ locate zip2john

6.3 Поддерживаемые методы шифрования

zip2john поддерживает различные методы шифрования ZIP, включая:

  • Традиционный ZipCrypto
  • AES-128 и AES-256
  • Другие распространенные методы сжатия ZIP
